# GitHub の使い方


1. リポジトリからデータを取得
$ git clone https://github.com/DZGK/Hello-World

2. リモートリポジトリを追加
usage: git remote add <shortname> <url>

$ git remote add hw https://github.com/DZGK/Hello-World




##
1. リモートリポジトリを追加
git add *


3. commit





4. リモートリポジトリへ push
git push


##
$ git remote -v
